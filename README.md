# 🔐 Password Manager

A secure, full-stack password manager built with **Next.js**, **Supabase**, and **Tailwind CSS**. Store, organize, and manage your passwords with built-in strength checking and a password generator.

**🔗 Live Demo:** [password-manager-app-snowy.vercel.app](https://password-manager-app-snowy.vercel.app)

---

## ✨ Features

- **User Authentication** — secure signup and login powered by Supabase Auth
- **Password Vault** — add, view, search, filter, and delete saved passwords
